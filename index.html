About the Game

Brick Bash is a classic arcade game where your goal is to destroy all the bricks on the screen by bouncing a ball off a paddle. Test your reflexes and aim to score high!

  • Use the left and right arrow keys to control the paddle.
  • Break all the bricks to win the game.

Game Features

  • Smooth gameplay with paddle and ball physics.
  • Dynamic ball gradient for a visually appealing design.
  • Brick collision detection to handle brick removal and score updates.
  • Lives counter: Players start with 3 lives and lose one when the ball hits the bottom.
  • Score tracking
  • Victory: Congratulatory message when all bricks are cleared.
  • Game Over: Displays the final score when lives are depleted.
  • Paddle movement using arrow keys for intuitive control.
